### Projeto desenvolvido para Aula de Python para Data Engineer

A ideia do projeto é construir um ETL simples, que lê alguns arquivos da pasta dados, faz uma transformação e salva na pasta resultados.

Libs necessárias:
- PyYAML
- Bibtexparser
- Pandas
